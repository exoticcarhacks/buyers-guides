## 5. Pre-Purchase Inspection (PPI) & Known Issues
A thorough Pre-Purchase Inspection (PPI) by a Ferrari specialist is highly recommended before any purchase.

### Known Problems
| Issue | What To Know/Check |
|---|---|
| Wonky starter motors | can result in unreliable starting performance, especially after the engine is shut off after being driven for a few dozen miles. Accompanied by electrical warning errors. Fix involves replacing starter motor and battery connector cable. Cost: Not specified, but repairs can be expensive. |
| Rough-shifting transmissions | 7DCL750 dual-clutch gearbox may leak fluid or experience speed sensor failure. Speed sensor failure fix: approximately $1,000. DCT leak rebuild: around $10,000. DCT replacement: $30,000-$40,000. |
| Peeling paint | occurs mostly on the rear mesh, possibly due to overheating from the exhaust. Simple fix, covered under factory warranty. |
| Leather shrinkage on dashboard | description and cost — Not specified. |
| Leather problems in rear sunscreen | description and cost — Not specified. |
| Creaking from torque bolts in rear driveshaft | description and cost — Not specified. |
| Corrosion at rear mesh | covered under warranty or extended Ferrari warranty. |
| Hot starting issue/Lazy start with Warm engine | various fixes including updated alternator kit, custom modded starting cable, or extra GND cable. Cost: Not specified. |
| Lifter handle on racing seats prone to breaking | description and cost — Not specified. |
